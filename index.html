¿Cómo funciona?

  1. Haz clic en "Activar Cámara y Escanear" para iniciar
  2. Permite el acceso a la cámara cuando tu navegador lo solicite
  3. La cámara se activará por 3 segundos para escanear
  4. Apunta hacia el código de barras del producto Walmart
  5. El sistema identificará el producto automáticamente
  6. Verás el EcoPuntos y el impacto ambiental detallado
